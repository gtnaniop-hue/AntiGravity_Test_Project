# FitPulse - Fitness Tracker

A full-stack fitness tracking application built with MERN stack.

## Tech Stack
- **Frontend**: React, Vite, TypeScript, Tailwind CSS, ShadCN UI
- **Backend**: Node.js, Express, MongoDB
- **Tools**: Recharts, Zustand, React Hook Form

## Getting Started

### Backend
1. Navigate to `backend` folder.
2. Run `npm install`.
3. Create a `.env` file with `MONGO_URI` and `JWT_SECRET`.
4. Run `npm run dev`.

### Frontend
1. Navigate to `frontend` folder.
2. Run `npm install`.
3. Run `npm run dev`.
